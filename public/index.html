STRING CLASS EXTENSIONS

What is this?

String Class Extensions are methods that are added to the existing String object prototype to add more functionality. They also modify some of the existing string class methods.

Click the button below to see the list of the methods that extends the string class.

view

Click the add icon to see more info about the method.

  • addhasVowel

    Returns true if the string contains vowels

  • addtoUpper

    Returns the String in question but with all characters in upper cases as applicable.

  • addtoLower

    Returns the String in question but with all characters in their lower cases as applicable

  • adducFirst

    Returns the String in question but changes the First Character to an Upper case

  • addisQuestion

    Return true if the string is a question (ending with a question mark).

  • addremoveSpace
